Question-1: What are the different ways to select an element in the DOM?

We can select html elements in dom in different ways:
1. getElementById()
2. querySelector()
3. querySelectorAll()
4. getElementsByClassName()
5. getElementsByTagName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : We use innerHTML to fetch HTML element which includes tag, we can also set the HTML element using innerHTML.
innerText : We use innerText to fetch text element which is inside an HTML element, but it does not include the hidden texts, Again on the other hand, we can also set the text of that element using innerText.
textContent : We use textContent to fetch text inside an HTML element which includes hidden text also we can also set the text using textContent.

Question-3: What is event delegation in the DOM?

The technique in JS in which the parent element handles the events for its child element, even if the child nodes are later added. This is known as EVENT DELEGATION.

Question-4: What is event bubbling in the DOM?

The process in JS where the event starts from target element and eventually moves to the up through its parent element in DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an element : const elem = document.createElement("h1");
elem.innerText = 'Hello World';
Adding an Element to the Page: document.body.appendChild(elem);
Removing an Element: elem.remove();
